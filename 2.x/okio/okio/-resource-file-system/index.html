class ResourceFileSystem : FileSystem
Content copied to clipboard

A file system exposing Java classpath resources. It is equivalent to the files returned by ClassLoader.getResource.

Both metadata and file listings are best effort, and will work better for local project paths. The file system does not handle merging of multiple paths from difference resources like overlapping .jar files.

ResourceFileSystem excludes .class files from calls to list, but can read them in calls to read and source.
